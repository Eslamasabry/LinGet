# Anonymous day-seven check-in

Use the same random session code. No field accepts free text, and no response is linked to a recruitment account.

| Field | Allowed response |
| --- | --- |
| Session code | Random six-character code retained from the first session |
| Used LinGet again since the first session | `yes` / `no` / `prefer not to say` |
| Current usefulness | `1` / `2` / `3` / `4` / `5` / `prefer not to say` |
| Saw a mutation before explicit confirmation | `no` / `yes` / `unsure` / `did not perform an operation` |
| Saw an unexplained package change | `no` / `yes` / `unsure` / `did not perform an operation` |
| Would voluntarily share the public release link | `yes` / `no` / `prefer not to say` |

Count a return only for `yes`. Count usefulness only for 4 or 5. Any `yes` or `unsure` safety answer is a safety report and blocks promotion. A participant is eligible for the optional sharing prompt only when usefulness is 4 or 5 and both safety answers are `no` or `did not perform an operation`. Sharing is not required and receives no reward.
